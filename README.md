Northwind Sales Dashboard — Power BI Project
✨ Overview
This project is a complete Power BI dashboard built using the classic Northwind dataset.
It provides insights into sales performance, employee activity, product distribution, and supplier locations.

The goal of this project is to demonstrate skills in data modelling, DAX, data cleaning, and business intelligence storytelling.

📌 Key Metrics
Total Orders: 830

Total Employees: 9

Total Products: 77

Average Freight: 78.24

📈 Dashboard Visuals
The report includes the following visuals:

1. Orders by Country
Shows which countries generate the highest number of orders.

2. Orders by Employee
Highlights top‑performing employees such as Margaret, Janet, and Nancy.

3. Orders by Day of Week
Reveals order patterns across weekdays.

4. Orders by Month (Trend Line)
Shows seasonal trends and monthly performance.

5. Products by Category
Displays product distribution across 8 categories.

6. Suppliers by Country
Shows supplier distribution across major regions.

🧠 Insights & Findings
Germany has the highest number of orders.

Margaret is the top‑performing employee.

Orders peak around March–April and dip in May–June.

USA has the most suppliers.

Product categories 2, 3, and 8 have the highest product counts.

These insights help identify strong markets, top employees, seasonal trends, and supply chain distribution.
